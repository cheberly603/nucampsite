How to Install Discord on PC or Mac


Discord Download

  1. Open your preferred web browser
  2. Visit: Discord.com
  3. Open the "hamburger" menu (three stacked horizontal lines)
  4. Select "Download"
  5. Choose the download button relevant for your computer (windows/iOS/Mac/Android)
    • this should open up a file location on your computer. Specify the file you would like to save the download into and select "save"

The First Step is Complete!


Getting Started

  1. Find the file location for Discord and double click the icon to open, or right-click/control-click(mac) to find options to open.
  2. Once installed, you will be prompted to create a Discord account! Follow the on screen prompts and you'll be on your way to finding your Discord community in no time!
